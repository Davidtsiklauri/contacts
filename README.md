#### 1)Clone Project

```sh
git clone https://github.com/Davidtsiklauri/contacts.git
```
#### 2) CD 

```sh
cd contacts
```

#### 3) Install Dependencies

```sh
npm run install-dependencies
```

#### 4) Run Back-End

```sh
npm run start-backend
```

#### 4) Open New Terminal And Run

```sh
npm run start-front-end
```


#### 5) Open chrome on 
```sh
 http://localhost:8080/
```
